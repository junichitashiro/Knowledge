# セキュリティ監視構成図

---

## SOCセキュリティ監視構成

```mermaid
flowchart LR

User[ユーザー] --> Internet

subgraph 外部
    Internet
end

subgraph 社内ネットワーク
    FW[Firewall]
    IDS[IDS/IPS]
    WAF[WAF]
end

subgraph サーバ群
    Web[Web Server]
    App[Application Server]
    DB[(Database)]
end

subgraph 監視基盤
    LogAgent[Log Agent]
    SIEM[SIEM]
    SOAR[SOAR]
end

subgraph SOC
    Analyst[SOC Analyst]
    Ticket[Incident Ticket]
end

Internet --> FW
FW --> WAF
WAF --> Web

Web --> App
App --> DB

Web --> LogAgent
App --> LogAgent
FW --> LogAgent
IDS --> LogAgent

LogAgent --> SIEM
SIEM --> SOAR

SIEM --> Analyst
Analyst --> Ticket
```

---

## SOC構成の役割

| 層       | 役割                 |
| -------- | -------------------- |
| ログ収集 | 各機器からログを収集 |
| 分析     | SIEMで相関分析       |
| 自動対応 | SOARで自動化         |
| 運用     | SOCアナリストが対応  |

---

## SOCログ収集対象

SOCでは以下のログが必須

### ネットワーク機器

* Firewall
* IDS / IPS
* WAF
* VPN

### サーバ

* Web Server
* Application Server
* Database

### エンドポイント

* EDR
* Windows Event Log
* Linux Syslog

### クラウド

* AWS CloudTrail
* Azure Monitor
* GCP AuditLog

## 実務SOC構成

最近は **EDR + SIEM + SOAR + Threat Intelligence** が主流

```mermaid
flowchart LR

subgraph Endpoint
    EDR[EDR Agent]
end

subgraph Network
    FW[Firewall]
    IDS[IDS/IPS]
end

subgraph Cloud
    CloudLog[Cloud Audit Log]
end

subgraph LogPlatform
    LogCollector[Log Collector]
    SIEM[SIEM]
end

subgraph Automation
    SOAR[SOAR]
end

subgraph ThreatIntel
    TI[Threat Intelligence]
end

subgraph SOC
    Analyst[SOC Analyst]
end

EDR --> LogCollector
FW --> LogCollector
IDS --> LogCollector
CloudLog --> LogCollector

LogCollector --> SIEM

TI --> SIEM

SIEM --> SOAR
SIEM --> Analyst

SOAR --> Analyst
```
