# 文字列の置換

***

## 文字列を置換する

* __before__ を __after__ に置換する

  ```PowerShell
  $text = "text-before"
  $text -replace "before","after"
  ```

  ```PowerShell
  $text = "text-before"
  $text.replace("before","after")
  ```

  実行結果

  > text-after

* -replaceは __大文字と小文字の区別をしない__ ので注意する

  ```PowerShell
  $text = "Memo"
  $text -replace "M","-"
  ```

  実行結果

  > -e-o

* __-creplace__ は大文字と小文字の区別をして置換できる

  ```PowerShell
  $text = "Memo"
  $text -creplace "M","-"
  $text.replace("M","-")
  ```

  実行結果

  > -emo  
    -emo
