# カレントディレクトリの取得

***

## カレントディレクトリを取得して変数に格納する

* コマンドから実行する場合

  ```cmd
  for /f %i in ('chdir') do set DIR=%i
  echo %DIR%
  ```

  実行結果

  > C:\Users\username

* バッチの記述方法

  ```bat
  for /f %%i in ('chdir') do set DIR=%%i
  echo %DIR%
  ```

* 以下のコマンドで取得した場合、パスの末尾に "\\" が入る

  ```bat
  set DIR=%~dp0
  echo %DIR%
  ```

  実行結果

  > C:\Users\username\
