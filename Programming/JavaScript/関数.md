# 関数

---

## 関数の作成

* 引数なし、戻り値なし

  ```javascript
  function sayHi() {
    console.log('Hi!');
  }

  sayHi();
  ```

  実行結果

  >  Hi!

* 引数あり、戻り値なし

  ```javascript
  function sayHi(name) {
    console.log(`Hi! ${name}`);
  }

  sayHi('Taro');
  ```

  実行結果

  > Hi! Taro

* 引数あり、戻り値あり

  ```javascript
  function square(num) {
    const result = num * num;
    return result;
  }

  square(10);
  ```

  実行結果

  > 100

## 関数式の作成

* 引数なし、戻り値なし

  ```javascript
  const sayHi = function() {
    console.log('Hi!');
  }

  sayHi();
  ```

  実行結果

  > Hi!

---

## アロー関数式の作成

* 引数なし、戻り値なし

  ```javascript
  const sayHi = () => {
    console.log('Hi!');
  }

  sayHi();
  ```

  実行結果

  > Hi!

* 引数あり、戻り値あり

  ```javascript
  const multi = (a, b) => {
    return a * b;
  }

  multi(3, 5);
  ```

  実行結果

  > 15
