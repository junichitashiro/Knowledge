# ファイルの作成

---

## 空ファイルの作成

* 0バイトのファイルを作成する

  ```PowerShell
  New-Item -ItemType file null.txt
  ```

* 現在時刻の空ファイルを作成する１

  ```PowerShell
  $fileName = Get-Date -Format "yyyyMMddhhmmss"
  Out-File "$fileName.txt"
  ```

* 現在時刻の空ファイルを作成する２

  ```PowerShell
  $fileName = Get-Date -Format "yyyyMMddhhmmss"
  New-Item -ItemType file "$fileName.txt"
  ```
