# 文字列の分割

---

## 文字列を分割する

### 分割する文字列を指定しないと スペース で分割される

```PowerShell
$text = "text split"
-split $text
```

```PowerShell
$text = "text split"
$text.split()
```

> text  
  split

### ”,”で文字列を分割する

```PowerShell
$text = "text,split"
$text -split ","
```

```PowerShell
$text = "text,split"
$text.split(",")
```

> text  
  split

### ”c”または"f" で文字列を分割する

```PowerShell
$text = "abcdefg"
$text -split "[cf]"
```

```PowerShell
$text = "abcdefg"
$text.split("[cf]")
```

> ab  
  de  
  g
