# 文字列の置換

---

## 文字列を置換する

* **before** を **after** に置換する

  ```PowerShell
  $text = "text-before"
  $text -replace "before","after"
  ```

  ```PowerShell
  $text = "text-before"
  $text.replace("before","after")
  ```

  実行結果

  > text-after

* -replaceは **大文字と小文字の区別をしない** ので注意する

  ```PowerShell
  $text = "Memo"
  $text -replace "M","-"
  ```

  実行結果

  > -e-o

* **-creplace** は大文字と小文字の区別をして置換できる

  ```PowerShell
  $text = "Memo"
  $text -creplace "M","-"
  $text.replace("M","-")
  ```

  実行結果

  > -emo  
    -emo
