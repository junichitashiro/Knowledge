# 文字列の置換

---

## 文字列を置換する

### before を after に置換する

```PowerShell
$text = "text-before"
$text -replace "before","after"
```

```PowerShell
$text = "text-before"
$text.replace("before","after")
```

> text-after

### -replaceは 大文字と小文字の区別をしない ので注意する

```PowerShell
$text = "Memo"
$text -replace "M","-"
```

> -e-o

### -creplace は大文字と小文字の区別をして置換できる

```PowerShell
$text = "Memo"
$text -creplace "M","-"
$text.replace("M","-")
```

> -emo  
  -emo
