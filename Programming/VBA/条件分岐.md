# 条件分岐

## Select - Case

### 文字列を判定して条件を分岐する

* 変数の値を評価して処理を変える

```vb
Sub SelectCase1()

    Dim drink
    drink = "coffee"

    Select Case drink
        Case "coffee"
            MsgBox ("コーヒーです")
        Case "tea"
            MsgBox ("紅茶です")
        Case "juice"
            MsgBox ("ジュースです")
        Case Else
            MsgBox ("どれでもありません")
    End Select

End Sub
```

### 変数が特定の値の場合に処理を実行する

* 変数が1,3,5の時にメッセージボックスで表示する

```vb
Sub SelectCase2()

    Dim i As Long

    For i = 1 To 7
        Select Case i
            Case 1, 3, 5
                MsgBox i
        End Select
    Next

End Sub
```
