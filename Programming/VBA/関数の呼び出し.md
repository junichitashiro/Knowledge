# 関数の呼び出し

## 引数と戻り値の設定

* calledFuncに引数1,2,3を渡し、calledFuncから戻り値1,2,3を受け取る

  ```vb
  Sub FunctionTest()

      i = "引数1"
      j = "引数2"
      k = "引数3"

      Call CalledFunc(i, j, k)

      MsgBox (i)
      MsgBox (j)
      MsgBox (k)

  End Sub
  ```

* 呼び出される関数の定義

  ```vb
  Function CalledFunc(i, j, k)

      MsgBox (i)
      MsgBox (j)
      MsgBox (k)

      i = "戻り値1"
      j = "戻り値2"
      k = "戻り値3"

  End Function
  ```

  実行結果

  > 引数1  
    引数2  
    引数3  
  >
  > 戻り値1  
    戻り値2  
    戻り値3
