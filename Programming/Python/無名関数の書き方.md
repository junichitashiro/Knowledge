# 無名関数の書き方

## 通常の関数

* aとbを足すだけの関数

```python
def add(a, b):
    return a + b

add(3, 4)
```

### 実行結果

> 7

---

## lambdaを使った書き方１

* (lambda 引数: 返り値)(値)

```python
(lambda a, b: a + b)(3, 4)
```

### 実行結果

> 7

---

## lambdaを使った書き方２

* lambdaを変数に格納する書き方

```python
add = lambda a, b: a + b

add(3,4)
```

### 実行結果

> 7
