# ファイルの検索

## ファイル名で検索する

### .log を指定してカレントディレクトリ配下を検索する

```cmd
dir /a-d /b /s | findstr \.log
```

---

## ファイル中の文字列を検索する

### 文字列 critical が error.log 中に含まれるか検索する

```cmd
findstr "critical" error.log
```

### error.log ファイルを読み込んで critical が含まれる行を表示する

```cmd
for /f %i in ("error.log") do (findstr "critical" %i)
```
