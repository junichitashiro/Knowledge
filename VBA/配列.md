# 配列

## 固定長配列

* 固定長の配列に値を格納する

  ```vb
  Sub Array1()

      Dim arr(2)

      arr(0) = "No1,"
      arr(1) = "No2,"
      arr(2) = "No3"

      MsgBox Join(arr)

  End Sub
  ```

* 実行結果

  > No1,No2,No3

***

## 可変長配列

* 可変長の配列に値を格納する１

  ```vb
  Sub Array2()

      Dim arr()

      ReDim arr(0)
      arr(0) = "No1,"

      ReDim arr(1)
      arr(1) = "No2,"

      ReDim arr(2)
      arr(2) = "No3"

      MsgBox Join(arr)

  End Sub
  ```

* 実行結果  
  ※"No1,"と"No2,"が再定義により削除されている

  > No3

***

* 可変長の配列に値を格納する２

  ```vb
  Sub Array3()

      Dim arr()

      ReDim Preserve arr(0)
      arr(0) = "No1,"

      ReDim Preserve arr(1)
      arr(1) = "No2,"

      ReDim Preserve arr(2)
      arr(2) = "No3"

      MsgBox Join(arr)

  End Sub
  ```

* 実行結果  
  ※再定義をしても"No1,"と"No2,"が削除されていない

  > No1,No2,No3
