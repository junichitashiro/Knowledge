# 配列  

## 固定長配列  
* 固定長の配列に値を格納する
```vb
Sub Array1()
    Dim Arr(2)

    Arr(0) = "No1,"
    Arr(1) = "No2,"
    Arr(2) = "No3"

    MsgBox Join(Arr)
End Sub
```
* 実行結果
> No1,No2,No3

## 可変長配列
* 可変長の配列に値を格納する１
```vb
Sub Array2()
    Dim Arr()

    ReDim Arr(0)
    Arr(0) = "No1,"

    ReDim Arr(1)
    Arr(1) = "No2,"

    ReDim Arr(2)
    Arr(2) = "No3"

    MsgBox Join(Arr)
End Sub
```
* 実行結果  
※"No1,"と"No2,"が再定義により削除されている
> No3

* 可変長の配列に値を格納する２
```vb
Sub Array3()
    Dim Arr()

    ReDim Preserve Arr(0)
    Arr(0) = "No1,"

    ReDim Preserve Arr(1)
    Arr(1) = "No2,"

    ReDim Preserve Arr(2)
    Arr(2) = "No3"

    MsgBox Join(Arr)
End Sub
```
* 実行結果  
※再定義をしても"No1,"と"No2,"が削除されていない
> No1,No2,No3
